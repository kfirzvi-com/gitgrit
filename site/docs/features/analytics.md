# Analytics Dashboard

The Analytics tab on your dashboard provides insights into compliance trends across your workspace.

## Summary stats

At the top, you'll see counts for:

- **Projects** — total repositories monitored
- **Policies** — active policies
- **Executions (30d)** — policy runs in the last 30 days
- **Labels** — policy labels configured

## Charts

### Compliance Trend

A 30-day line chart showing the average compliance score across all projects and policies. Helps you spot regressions or improvements over time.

### Scores by Label

A horizontal bar chart showing average scores grouped by policy label (e.g., security, quality, ci-cd). Quickly identify which areas need attention.

## Tables

### Project Compliance

All projects sorted by compliance score (lowest first), with language badges and a progress bar.

### Top Failing Policies

The 5 policies with the highest failure rates, showing failure count, total executions, and fail rate percentage.

## Demo mode

Toggle **Demo data** to preview how the dashboard looks with realistic data — useful for evaluating the platform before you have real policy executions.
