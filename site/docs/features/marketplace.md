# Policy Marketplace

The marketplace provides pre-built policy templates that you can install into your workspace with one click.

## Browsing

Go to **Marketplace** from the navigation bar. You'll see:

- **Packs** — curated bundles of related policies (e.g., "Repository Hygiene", "Security Essentials")
- **Individual policies** — browse all available policies with labels, author, and version info

## Installing

Click **Install** on any policy or **Install All** on a pack. This creates a copy in your workspace that you can customize freely.

Installed policies show a green "Installed" badge in the marketplace.

## Customizing

After installing, the policy is yours to edit — change the code, adjust criteria, modify labels. The original marketplace policy is tracked as a source, so you can see when updates are available.

## Updates

When a marketplace policy is updated (new version), you'll see an "Update available" badge on your policy list and detail pages. Updating overwrites your code with the new version — a confirmation is shown before proceeding.
